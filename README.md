# Formation Performance Java : Diagnostic & Résolution

Ce dépôt contient le site vitrine (statique) présentant ma formation spécialisée dans le diagnostic de performance Java en production.

## 🎯 Objectif de la formation

L'objectif est de rendre les équipes de développement autonomes dans l'identification et la résolution de problèmes critiques :

- **CPU** : Surcharges inexpliquées et optimisation.
- **Mémoire** : Détection et correction de fuites (Memory Leaks & Soft Leaks).
- **Threads** : Analyse de Deadlocks, Lock Contention et Thread Leaks.

## 🛠️ Contenu du projet

Le site est construit de manière minimaliste et performante pour un chargement instantané :

- `index.html` : Structure sémantique et contenu pédagogique.
- `styles.css` : Design moderne, compact et responsive (optimisé pour mobile).
- **Stack** : HTML5, CSS3, Lucide Icons (via CDN).

## 🚀 Déploiement

Ce site est optimisé pour être hébergé sur **GitHub Pages**.

1. Clonez le dépôt.
2. Activez GitHub Pages dans les réglages (`Settings > Pages`).
3. Le site est en ligne.
